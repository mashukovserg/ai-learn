# AI-Learn Platform

Платформа для изучения AI с интерактивной теорией и заданиями, построенная на Next.js App Router + FastAPI бэкенде. Сочетает структуру комнат/заданий в стиле TryHackMe (прогрессивные комнаты, практические челленджи) с геймификацией в стиле Duolingo (очки, серии, прогресс-бары, радар навыков, короткие уроки с мгновенной обратной связью).

**Демо:** [ai-learning-platform-murex.vercel.app](https://ai-learning-platform-murex.vercel.app)

## Текущая архитектура

### Фронтенд (Next.js)
- `src/app/[lang]/...` — локализованные маршруты.
- `src/components/AppShell.tsx` — общий лейаут (оборачивает всё в `AuthProvider`).
- `src/components/Sidebar.tsx` + `src/components/Navbar.tsx` — навигация (показывает реальные данные пользователя при авторизации).
- `src/data/glossary.ts` — централизованный словарь терминов для теоретических комнат.
- `src/components/Term.tsx` — hover-компонент, показывающий определение термина прямо в тексте.
- `src/hooks/useProgress.ts` — сохранение прогресса (API при авторизации, localStorage для гостей).
- `src/hooks/useAuth.tsx` — контекст авторизации (login/signup/logout, профиль из `/me`).
- `src/components/RoomStatusBadge.tsx` + `DashboardProgress.tsx` — отображение прогресса.
- `src/dictionaries/{en,ru}.json` + `src/dictionaries/get-dictionary.ts` — локализованные строки UI.
- `next.config.ts` проксирует запросы `/api/*` на бэкенд `http://localhost:8000`.

### Бэкенд (FastAPI)
- `backend/app/main.py` — FastAPI-приложение с CORS и роутерами.
- `backend/app/database.py` — асинхронный SQLAlchemy engine + сессия.
- `backend/app/db/models/` — ORM-модели: `UserORM`, `AuthSessionORM`, `UserProgressORM`, `AgentTaskORM`, `AgentRunORM`, `AgentKnowledgeItemORM`.
- `backend/app/db/repositories/` — паттерн Repository (коллекции + репозитории объектов).
- `backend/app/api/auth/` — регистрация, вход, выход с bcrypt + JWT cookies.
- `backend/app/api/progress/` — GET/POST прогресса по комнатам, логика очков/серии.
- `backend/app/api/users/` — GET `/me` для профиля + статистики.
- `backend/app/api/agent/` — API Agent Ops (минимальный CRUD очереди, запуск цикла, лента запусков и лента базы знаний).
- `backend/app/api/dependencies/` — `DBSessionDep`, `CurrentUserIDDep` (cookie → сессия → user_id).
- `backend/alembic/` — асинхронные миграции Alembic для PostgreSQL.

## Известные ограничения
- Ссылки в сайдбаре `/${lang}/compete` и `/${lang}/leaderboard` присутствуют в UI, но страницы ещё не реализованы.
- Метаданные комнат дублируются между страницами (дашборд/комнаты/пути определяют свои массивы).
- В Agent Ops запуск цикла в MVP выполняется вручную (`POST /api/agent/cycle/run`); периодический scheduler пока не подключен.

## Правила контента
- **Стиль (прагматичный обучающий нарратив):** использовать концептуальные аналогии (скаффолдинг), поддерживать разговорный тон для снижения «технической тревожности», давать контекстное обоснование важности темы, придерживаться чёткой структуры с дорожной картой.
- **Правило локализации:** весь пользовательский контент уроков и заданий должен быть доступен на обоих языках (без блоков только на EN или только на RU).
- **Правило покрытия заданий:** каждое задание должно быть решаемо на основе текста урока на обоих языках.
- **Правило явного маппинга:** если задание требует классификации (например, «Выберите модели, которые НЕ принадлежат компаниям из США»), урок должен явно содержать этот маппинг в тексте теории.
- **Правило подсказок к терминам:** каждая комната должна содержать всплывающие подсказки (или эквивалентные пояснения) для сложных технических терминов, чтобы начинающие могли понять контент, не покидая страницу.

## Система глоссария
- **Где хранится:** термины глоссария лежат в `src/data/glossary.ts` в виде типизированной карты `GLOSSARY`.
- **Как работает:** `src/components/Term.tsx` берёт термин по `id`, выбирает нужный язык (`ru/en`) и показывает определение во всплывающей подсказке при наведении.
- **Как использовать в теории:** вставляйте `<Term id="..." lang={lang}>...</Term>` прямо в текст урока. Если `children` не переданы, компонент возьмёт стандартную подпись из `GLOSSARY.term`.

## Разработка

### Только фронтенд (гостевой режим, прогресс в localStorage)
```bash
npm install
npm run dev
```
Откройте `http://localhost:3000` (перенаправит на `/ru`).

### Полный стек (с авторизацией и сохранением прогресса на сервере)
```bash
# Терминал 1: Запуск PostgreSQL
cd backend && docker compose up -d db

# Терминал 2: Миграции + запуск бэкенда
cd backend && uv sync && .venv/bin/alembic upgrade head && .venv/bin/python -m app.main

# Терминал 3: Запуск фронтенда
npm run dev
```
Бэкенд работает на `:8000`, фронтенд автоматически проксирует `/api/*` на него.
