# O*NET-калибровка экспертного рейтинга профессий

Сверяет экспертные баллы модели v3.0 (три фактора, 1–10) с объективным прокси из данных
O*NET и выносит расхождения на ревизию. Полное методическое описание и выводы —
`docs/onet-calibration-memo.md`.

## Запуск

```bash
cd scripts/onet_calibration
python3 calibrate.py
```

Python 3, только стандартная библиотека. `openpyxl` опционален (для xlsx; без него — CSV).

## Ключевой вывод текущего прогона

Загруженный файл O*NET **Skills** оказался **слабым прокси** для всех трёх факторов
(Spearman: F1=0.28, F2=0.15, F3=0.29). Для корректной калибровки нужны файлы O*NET
**Work Context** и **Abilities** — какие именно переменные брать, см. §4 методички.
Гарнитура готова: замените корзины в `factor_baskets.py` на имена этих переменных.

## Структура

| Файл | Назначение |
|---|---|
| `calibrate.py` | расчёт прокси, ранговая калибровка, Spearman-диагностика |
| `factor_baskets.py` | корзины навыков → факторы (конфиг, правится вручную) |
| `mapping_ru_onet.csv` | справочник RU-профессия → O*NET-SOC (ручная проверка) |
| `expert_scores.csv` | экспертные баллы f1/f2/f3 (из Google Sheet) |
| `data/onet_skills.csv` | исходные данные O*NET Skills |
| `out/` | результаты (calibration, correlations, unmapped) |
