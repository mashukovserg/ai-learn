# Референс: re.alisa.sh (Alisa Esage)

- **URL:** https://re.alisa.sh/
- **Дата съёма:** 2026-07-12 (полный CSS извлечён из живой страницы, ~9 КБ)
- **Что это:** персональный сайт исследовательницы безопасности — одна страница: hero-образ (чёрный монолит), ссылка на свежее эссе, два «бренда» строками, соцссылки, форма запроса доступа
- **Зачем нам:** эталон сдержанности для тёмной темы «Терминал»; источник приёмов типографики и иерархии

## Извлечённые токены и приёмы (фактаж)

### Палитра (CSS-переменные сайта)

```css
--bg: #07080d;                          /* почти чёрный с синим подтоном */
--text: rgba(210, 215, 225, 0.88);      /* основной текст */
--text-mid: rgba(190, 195, 210, 0.6);   /* второй уровень */
--text-dim: rgba(160, 165, 180, 0.38);  /* мета-лейблы, подписи */
--rule: rgba(180, 190, 210, 0.1);       /* линии-разделители */
--rule-bright: rgba(180, 190, 210, 0.18);
--accent: #0C2073;                      /* тёмно-синий; ТОЛЬКО hover-подчёркивания */
```

### Шрифты

```css
/* Google Fonts: два семейства, четыре начертания суммарно */
"Cormorant Garamond" (300, 400, italic 300/400) — названия, фразы, курсивные ссылки
"Share Tech Mono" — мета-лейблы: 8–10px, letter-spacing 2–5px, uppercase
```

### Ключевые CSS-приёмы

1. **Плёночное зерно без запросов** — `body::after` c inline-SVG `feTurbulence`, `opacity: .35`, `pointer-events: none`, `z-index: 999`.
2. **Вертикальные руллы** — `.rule-line`: `width: 1px; height: 56px; background: linear-gradient(transparent, rgba(180,195,225,.18), transparent)` — ритм между секциями вместо отступов.
3. **Списки строками** — `.brand-item`: `border-bottom: 1px solid var(--rule)` (+ border-top у первой), padding 28px 0; hover меняет только цвет текста.
4. **Формы «нижней границей»** — input: прозрачный фон, только `border-bottom`, моно-шрифт, центрированный текст; кнопка: прозрачная, рамка 1px, hover красит рамку в accent.
5. **Свечение как единственный эффект** — radial-gradient + `blur(24px)` под hero-образом.
6. **Ховеры** — только `color`/`border-color` за 0.2–0.25s; никаких transform/scale/теней.
7. `border-radius: 0` везде; всё в одну колонку, `max-width` контента ~520–580px.

## Разбор

**Эстетика:** «люксовый минимализм с кибер-оккультным настроением». Один экран-тотем (чёрный монолит), дальше — текстовые секции строками. Весь эффект держится на пяти приёмах:

1. **Alpha-иерархия вместо палитры.** Цветов почти нет — один светлый оттенок `rgb(210,215,225)` с разными прозрачностями: `--text` (.88) → `--text-mid` (.6) → `--text-dim` (.38) → линии (.1–.18). Иерархия читается мгновенно, а страница остаётся монолитной.
2. **Двухшрифтовый контраст «человеческое × машинное».** Cormorant Garamond (антиква 300, курсив) — для названий и фраз; Share Tech Mono 8–10px с разрядкой 2–5px и капсом — для мета-лейблов (`RECENT`, `RESEARCH · RECORD`). Это главный узнаваемый приём сайта.
3. **Акцент, который почти не виден.** `--accent: #0C2073` — глубокий тёмно-синий, применяется только к hover-подчёркиваниям. Никаких transform/scale/теней — ховеры меняют лишь цвет за 0.2s.
4. **Ритм вертикальными руллами.** Между секциями — не отступы, а 1px-линии высотой 48–56px с градиентом transparent→полупрозрачный→transparent.
5. **Плёночное зерно.** `body::after` с inline-SVG `feTurbulence` поверх всей страницы — дешёвый (0 запросов) киношный эффект.

Плюс детали: списки — не карточки, а строки с едва заметным border-bottom; формы — только нижняя граница, прозрачные кнопки с 1px-рамкой; `border-radius: 0` везде; свечение под кубом — единственный «спецэффект». И игровой паттерн «Access Denied → REQUEST» вместо обычной формы подписки.

## Что берём для платформы (совместимо с нашими гейтами)

| Приём | Как применить у нас |
|---|---|
| Моно-лейблы с разрядкой | Мета-информация комнат (категория, сложность, «ГЛАВА 1»-ярлыки над заголовками) — mono 10px, letter-spacing 3px, uppercase, dim. Идеально ложится на тему «Терминал» |
| Alpha-иерархия | Навести порядок в наших `text-neutral-300/400/500`: три ролевых уровня text/mid/dim — сейчас оттенки используются хаотично |
| Вертикальные руллы | Разделители между главами теории вместо пустых отступов — тонкий ритм без визуального шума (проходит anti-vibecode) |
| Списки строками | Список комнат внутри path-detail: строки с border-bottom и hover-цветом вместо карточек — спокойнее и плотнее |
| «Access Denied → REQUEST» | Наш locked-путь (Practical ML) оформить как терминальную заглушку `ACCESS DENIED` с кнопкой запроса — чистый TryHackMe-вайб, у нас будет смотреться роднее, чем у неё |
| Сдержанный hover-акцент | Подчёркивания ссылок тёмным emerald по hover вместо яркой заливки |

## Что не берём и почему

- **Антикву для текста** — serif для теории забракован владельцем (2026-07-09); Cormorant годится максимум для одной декоративной фразы на дашборде, и то спорно.
- **Тотальную центровку** — anti-vibecode gate требует left-align для длинного текста, и он прав: у Алисы центровка работает только потому, что текста почти нет.
- **Zero-radius** — у нас язык `rounded-xl`, ломать систему ради чужой эстетики не стоит.
- **Зерно поверх текста** — на платформе, где читают по 15 минут, утомит; максимум — на обложках/hero с opacity ~0.15.

## Мета-урок

Эффект достигается вычитанием — 9 КБ CSS, два шрифта, один акцентный цвет, ноль анимаций. Наши комнаты перегружены разнотипными блоками — курс на такую сдержанность у нас уже взят (anti-vibecode gate), этот сайт — хороший референс-ориентир.
